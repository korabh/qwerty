qwerty.vim
==============

A colorful, dark color scheme.

Use the plugin manager of your choice, here is an example for Vim-Plug:

`Plug 'korabh/qwerty'`
`:PlugInstall`

To preview:
```
syntax on
color qwerty
```
#### Terminal Background

If you would prefer to use your terminal's default background.
